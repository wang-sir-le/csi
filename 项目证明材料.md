# 无源 WiFi 人体微动与睡眠监测系统项目证明材料

## 可直接粘贴到“证明与影响力证明”输入框的文字

本人围绕“无源 WiFi 人体微动与睡眠监测系统”完成了从数据采集、模型训练到前后端原型展示的完整项目闭环。项目基于 ESP32 CSI 信道状态信息采集人体微动信号，已实现无人、静止、翻滚等状态数据采集，并通过特征构建、模型训练与实时识别流程，将 CSI 原始数据转化为可视化睡眠状态反馈。

在算法层面，项目包含 CSI 数据采集脚本、特征构建脚本、分组分析脚本、分类模型训练脚本和实时状态监测脚本，可完成数据读取、窗口化处理、特征提取、模型训练、实时推理和结果输出。在系统层面，已搭建 FastAPI 后端服务与 MySQL 数据库，数据库包含用户、作品、点赞、收藏、私信和检测会话等表；前端已实现实时看板、登录注册、个人主页、社区发布、图片/视频作品、帖子详情、点赞收藏、私信和开启/关闭检测等功能。

当前项目可在本地通过 `http://127.0.0.1:8000/` 访问演示。用户登录后可手动开启睡眠检测，系统实时读取串口 CSI 数据并更新看板状态；社区模块可发布睡眠监测经验和科普内容，个人主页展示作品、获赞、喜欢和检测状态。该项目已具备从硬件采集、机器学习识别、数据库持久化到 Web 产品展示的完整可运行原型，能够作为非接触式睡眠监测和人体微动识别方向的实践证明。
